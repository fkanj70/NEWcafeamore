# Café Amore Website Redesign ☕🇮🇹

A luxury website redesign concept for Café Amore, an Italian café located in Orléans, Ottawa.

This project was independently designed and developed as a modern web experience showcasing how a local café could elevate its online presence through premium design, storytelling, and user experience.

## Overview

The goal of this project was to create a high-end digital experience inspired by luxury hospitality brands while preserving the warmth and authenticity of an Italian family café.

Although the business ultimately chose to continue using their existing website, this project remains a portfolio piece demonstrating web design, front-end development, and user experience skills.

## Features

- Fully responsive design
- Luxury visual aesthetic
- Smooth scrolling animations
- Mobile navigation menu
- Customer reviews section
- Interactive location and contact section
- SEO-friendly metadata
- Accessibility improvements
- Modern typography and branding
- Optimized single-page experience

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Unsplash imagery

## Project Goals

- Improve the café's online presence
- Increase customer trust
- Create a premium brand image
- Showcase menu items and customer reviews
- Deliver a modern mobile-friendly experience
